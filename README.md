# Parkinson’s Disease Detection using Machine Learning

## 📌 Overview

This project uses machine learning algorithms to detect Parkinson’s Disease using voice data.

## ⚙️ Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy

## 🚀 How to Run

1. Open the notebook in Google Colab
2. Upload dataset if required
3. Run all cells

## 📊 Models Used

* Random Forest
* Extra Trees
* Logistic Regression (Meta Model)

## 📈 Result

Achieved high accuracy using ensemble learning techniques.
